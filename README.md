# 🚀 Mateus Oliveira - Portfólio

Meu portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e trajetória como estudante de Engenharia de Software e desenvolvedor em formação.

## 📖 Sobre o projeto

Este projeto foi criado para reunir meus principais projetos, experiências e tecnologias estudadas em um único lugar.

O objetivo é demonstrar minha evolução como desenvolvedor e servir como ponto central para recrutadores, colegas e empresas conhecerem meu trabalho.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## ✨ Funcionalidades

- Apresentação pessoal
- Seção sobre mim
- Projetos em destaque
- Timeline de aprendizado
- Links para GitHub e redes profissionais
- Design responsivo
- Modo escuro/claro
- Tradução Português/Inglês
